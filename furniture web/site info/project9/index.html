Список <тегов>


  • <p> - абзац
  • <i> - курсив
  • <b> - жирный шрифт
  • <q> - цитата, выделяется ковычками (quote)
  • <a> - ссылка, имеет атрибуты href и name
  • <dt> - термин
  • <li> - элемент списка
  • <br> - перенос строки
  • <dl> - список определений
  • <ol> - нумерованный список
  • <dd> - определение термина
  • <ul> - маркированный список
  • <div> - блочный элемент
  • <img> - изображение
  • <wbr> - допустимое место переноса.
  • <nav> - задает навигацию по сайту
  • <abbr> - абривиатура (title - пояснение)
  • <cite> - источник цитаты
  • <center> -выравнивает содержимое контейнера по центру относительно родительского элемента
  • <input> - позволяет создавать разные элементы интерфейса (оч. много атрибутов)
  • <button> - создает кнопку
  • <address> - информация об авторе
  • <blockquote> - блочная цитата
